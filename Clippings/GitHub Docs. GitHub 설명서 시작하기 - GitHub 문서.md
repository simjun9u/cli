## GitHub 설명서 시작하기

GitHub을 사용하여 소프트웨어를 처음으로 빌드하고, 배포하며, 유지 관리하는 방법을 습득하십시오. 제품을 탐색하고, 계정에 등록하고, 세계 최대의 개발 커뮤니티와 연결합니다.

### [여정 시작](https://docs.github.com/ko/free-pro-team@latest/get-started/start-your-journey)

GitHub이(가) 처음이신가요? 여기에서 기본 사항을 알아봅니다.

### [GitHub Copilot 사용하여 코딩하는 방법 알아보기](https://docs.github.com/ko/free-pro-team@latest/get-started/learning-to-code)

GitHub 및 Copilot이(가) 프로그래밍 기술을 구축하고, 더 나은 코드를 작성하고, 보안 프로젝트를 배송하는 데 어떻게 도움이 되는지 알아봅니다.

### [GitHub 계정 만들기](https://docs.github.com/ko/free-pro-team@latest/get-started/start-your-journey/creating-an-account-on-github)
